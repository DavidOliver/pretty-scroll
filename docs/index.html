Pretty Scroll

A library to make an element follow you as you scroll

Installation

npm

npm install pretty-scroll --save

or yarn

yarn add pretty-scroll

CDN

<script src="https://unpkg.com/pretty-scroll@1.0.2/js/pretty-scroll.js"></script>

Usage

JavaScript

new PrettyScroll('.js-sidebar', {
  container: '.js-container',
  breakpoint: 575
});

HTML

<div class="container js-container">
  <div class="main">
  <!-- main contents -->
  </div>
  <div class="sub">
    <div class="js-sidebar">
      <!-- sticky contents -->
    </div>
  </div>
</div>
